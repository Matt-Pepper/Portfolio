This website was made to look like the Windows95 operating system, I used HTML, SCSS, Javascript and JQuery.

This project is a Java console application, This was the biggest project I have created using Java which was fun to learn.

To get Minesweeper to work I needed to print the board to the screen, and also get use input for the size of the game and what move to make.
Once I had that implemented I needed to make sure the bombs would be randomized and the squares would have the correct content. Lastly I added all the game logic so the game would win and lose at the correct game state.

As an extra I made it so a users first move would always not be a bomb, I also made it so when a user selected a square with no bombs around it all squares with no bombs would be revealed up to the closet squares with numbers.

I really enjoyed creating this game because I needed to think through the project in a different way compared to building a website.
