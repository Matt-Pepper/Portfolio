This project is a Java console application and it was the biggest project I have created using Java, which was an exciting learning experience.

To create the Minesweeper game, I had to implement several elements such as printing the game board to the screen, getting user input for the size of the game and the next move. Additionally, I made sure that the bombs are placed randomly and each square displays the correct content. Lastly, I added the game logic to ensure that the game will end in the correct state.

I also added some extra features such as making sure that the first move of the user will never be a bomb, and if a user selects a square with no bombs around it, all squares with no bombs will be revealed up to the closest squares with numbers.

I enjoyed working on this project as it allowed me to think and approach it differently than when building a website.
